# MCP Servers - AI Education Tools

Collection of MCP (Model Context Protocol) servers for AI-powered educational applications.

## Projects

### 1. TrainBot MCP
AI-powered training course generator exposed as MCP tools.

### 2. StudyBuddy MCP
Educational assistant for Indian students (CBSE/ICSE/IGCSE).

## Usage

Each server can be used with Claude Desktop, Cursor, or any MCP-compatible client.

See individual project READMEs for setup instructions.
